# Strategy Narrative Structure (MUST)

## Purpose

Structure strategic communication so direction, rationale, and consequences remain explicit.

## Trigger

Use when strategic reasoning must be communicated in a way that preserves diagnosis, logic, and consequences.

## Rules (MUST)

- A strategic narrative must distinguish `why`, `how`, and `what` explicitly.
- Do not collapse diagnosis, prioritization, and measures into one undifferentiated story.
- Every narrative must make strategic trade-offs visible.
- Measures without underlying rationale are invalid.
- If material rationale elements are missing, mark the narrative as provisional.

## Fallback

- If diagnosis, prioritization logic, or consequence framing is incomplete, keep the narrative provisional and name the missing rationale elements.

## Minimal Output Contract

- why
- how
- what
- explicit trade-offs
- current result status (`preliminary|final`)
