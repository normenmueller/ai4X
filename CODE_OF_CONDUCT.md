# Code of Conduct

This project adopts the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Reporting

If you experience or witness unacceptable behavior, please report it through a private channel to the maintainers.
For security-related reports, use the process defined in `SECURITY.md`.
