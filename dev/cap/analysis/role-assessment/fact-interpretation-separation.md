# Fact vs Interpretation Separation (MUST)

## Purpose

Separate observed statements from inference, synthesis, and judgment.

## Trigger

Use when deriving interpretations, conclusions, or recommendations from source statements or observed evidence.

## Rules (MUST)

- Label observed source statements and derived interpretations explicitly.
- Do not present inference as if it were directly stated in the source.
- If an interpretation depends on an unstated assumption, surface that assumption.
- Recommendations must never bypass the fact-to-interpretation distinction.

## Fallback

- If the relevant evidence, scope, or comparison basis is incomplete, keep the result provisional and name the missing analytical basis.

## Minimal Output Contract

- observed statements
- derived interpretation
- open assumptions
