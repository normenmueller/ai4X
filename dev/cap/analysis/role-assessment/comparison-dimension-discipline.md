# Comparison Dimension Discipline (MUST)

## Purpose

Ensure that comparative analysis uses stable, decision-relevant dimensions instead of ad hoc framing.

## Trigger

Use when comparing alternatives, roles, proposals, or candidate options that need a stable evaluation frame.

## Rules (MUST)

- Define comparison dimensions before comparing alternatives or roles.
- Use the same dimensions across all compared objects unless an explicit scope break is declared.
- Keep missing values visible; do not silently replace unknowns with narrative judgment.
- Do not change dimensions mid-analysis to favor a preferred outcome.

## Fallback

- If the relevant evidence, scope, or comparison basis is incomplete, keep the result provisional and name the missing analytical basis.

## Minimal Output Contract

- comparison dimensions
- scoring or comparison rule
- missing values and comparability limits
