# arc42 Markdown Template Reference

This directory stores the official arc42 English Markdown template used as the structural reference for ai4X arc42 documents.

Source:
- https://arc42.org/download
- artifact: `arc42-template-EN-plain-markdown.zip`
- file: `arc42-template-EN.md`
- version: 9.0-EN (July 2025)

Do not edit `arc42-template-EN.md` manually.
When updating, replace it from the official arc42 distribution and keep this note in sync.
